HTML 101

This page will include what I like to call a "cheat sheet" of some common HTML rules to remember and also some basic HTML tags to get you started with creating your own web pages!

Common HTML Rules to Remember

  • HTML stands for HyperText Markup Language
  • HTML is used to create the structure of web pages
  • HTML elements are represented by tags, which are enclosed in angle brackets (e.g., <tagname>)
  • Most HTML elements have an opening tag and a closing tag (e.g., <tagname>content</tagname>)
  • HTML documents should have a proper structure, including <!DOCTYPE>, <html>, <head>, and <body> tags

Overall Structure of an HTML page

  • <!DOCTYPE html>: Declares the document type and version of HTML
  • <html>: The root element that contains all other HTML elements
  • <head>: Contains meta-information about the document, such as title, character set, and links to stylesheets
  • <body>: Contains the content of the document that is displayed in the web browser
  • <title>: Specifies the title of the document, which is displayed in the browser's title bar or tab
  • <link>: Links external resources to your HTML page such as your stylesheet or CSS page

Semantic HTML Tags

  • <header>: Represents the header section of a document or a section
  • <nav>: Defines a container for navigation links
  • <main>: Specifies the main content of a document
  • <section>: Represents a thematic grouping of content
  • <article>: Defines an independent, self-contained piece of content
  • <aside>: Represents content that is tangentially related to the main content
  • <footer>: Defines the footer section of a document or a section

The table below was created by me of some of my most frequently used HTML tags and other important ones you will probably use in the body of every HTML page

HTML Tag Description
<h1> to <h6> Defines HTML headings, with <h1> being the highest level and <h6> the lowest
<p> Defines a paragraph
<a> Defines a hyperlink
<img> Embeds an image in the document
<ul>, <ol>, <li> Defines unordered (ul) and ordered (ol) lists, with list items (li)
<div> Defines a division or section in an HTML document
<span> Defines a section in a document for styling purposes
<form>, <input>, <button> Defines an HTML form for user input, with input fields and buttons
<table>, <tr>, <td>, <th> Defines a table, with table rows (tr), table data cells (td), and table headers (th)
<br> Inserts a line break
