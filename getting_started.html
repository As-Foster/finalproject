Getting Started with HTML and CSS

Introduction

Here you will find instructions and resources to help you get started with code editors. My personal favorite code editor is Visual Studio Code (VS Code) because it is free, easy to use, and has a lot of extensions to help with web development.

Downloading VS Code

  1. Go to the official VS Code website: https://code.visualstudio.com/
  2. Click on the "Download" button for your operating system (Windows, macOS, or Linux).
  3. Once the download is complete, open the installer and follow the on-screen instructions to install VS Code.

Setting Up VS Code for Web Development

  1. Open VS Code after installation.
  2. Install the "Live Server" extension for real-time preview of your web pages:
    • Click on the Extensions icon in the sidebar (or press Ctrl+Shift+X).
    • Search for "Live Server" and click "Install".
  3. Create a new folder for your web projects and open it in VS Code:
    • Click on "File" > "Open Folder..." and select or create a folder.
  4. Create a new HTML file:
    • Click on "File" > "New File" and name it "index.html".
